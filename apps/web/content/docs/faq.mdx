<Accordion title="How do I arrange tiled windows with my mouse?">
  You can enable the `drag_tile_to_tile` option in your config. This allows you to drag a tiled window onto another to swap them.
  
  ```ini
  drag_tile_to_tile=1
  ```

</Accordion>

<Accordion title="Why is my background blurry?">
If you have enabled blur, it applies to transparent areas. If you simply want to save performance while keeping blur enabled for windows, ensure optimization is on:

```ini
blur_optimized=1
```

To disable blur entirely, set `blur=0`.
</Accordion>

<Accordion title="My games are lagging or stuttering.">
Try enabling **SyncObj** timeline support and **Adaptive Sync** (VRR) if your monitor supports it.

```ini
syncobj_enable=1
adaptive_sync=1
```

</Accordion>

<Accordion title="My games have high input latency.">
You can enable **Tearing** (similar to VSync off).

First, enable it globally:

```ini
allow_tearing=1
```

Then force it for your specific game:

```ini
windowrule=force_tearing:1,title:Counter-Strike 2
```

</Accordion>

<Accordion title="How do I use pipes `|` in spawn commands?">
The standard `spawn` command does not support shell pipes directly. You must use `spawn_shell` instead.

```ini
bind=SUPER,P,spawn_shell,echo "hello" | rofi -dmenu
```

</Accordion>
